# Portalfabriken Style Guide

## 4. Layout & Grid

The layout is structured to be both visually appealing and highly functional. It follows a consistent set of principles to ensure a cohesive user experience.

- **Full-Width Dark Theme**: The entire website uses a dark background, creating a consistent and immersive experience.
- **Centered Content**: The main content is centered within the viewport, with a maximum width to ensure readability on larger screens.
- **Responsive Grid**: The layout is built on a responsive grid system, allowing for flexible arrangements of content, such as the three-column layout in the "Ekosystem" section.
- **Generous Whitespace**: Ample whitespace is used around elements to reduce clutter and improve focus.
